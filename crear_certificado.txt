Genera la clave privada
openssl genrsa -out server.key 2048

Crea una solicitud de firma de certificado (CSR)
openssl req -new -key server.key -out server.csr 

Datos de ejemplo
Country Name (2 letter code) [AU]:CO
State or Province Name (full name) [Some-State]:Valle
Locality Name (eg, city) []:Cali
Organization Name (eg, company) [Internet Widgits Pty Ltd]:Ospedale
Organizational Unit Name (eg, section) []:Admin
Common Name (e.g. server FQDN or YOUR name) []:localhost
Email Address []:*********@********.com

Please enter the following 'extra' attributes
to be sent with your certificate request
A challenge password []:*********
An optional company name []:*********

Firma el certificado (autofirmado en este caso)
openssl req -x509 -newkey rsa:2048 -keyout rootCA.key -out rootCA.pem -days 825 -nodes -subj "/CN=My Root CA"
openssl x509 -req -in server.csr -CA rootCA.pem -CAkey rootCA.key -CAcreateserial -out server.crt -days 825 -sha256

Combina el certificado y la clave privada en un archivo PFX
openssl pkcs12 -export -out **********.pfx -inkey server.key -in server.crt -certfile rootCA.pem
